(inner) JobApplicantsPage(props)

Description:

  • JobApplicantsPage component

    This component renders a page displaying the list of job applicants. It includes navigation buttons
    to paginate through the applicants and a button to view detailed application information.

Example
return (
  <JobApplicantsPage
    Applicants={Applicants}
    isLoading={isLoading}
    onBack={onBack}
    onNext={onNext}
    onPrev={onPrev}
    hasMore={hasMore}
    currentPage={currentPage}
    jobId={jobId}
    onViewApplication={onViewApplication}
    selectedApplicant={selectedApplicant}
    onCloseApplicationDetails={onCloseApplicationDetails}
  />
)
Parameters:
Name Type Description
props Object

The properties object.

Properties
Name Type Description
Applicants Array

The array of applicant objects to display.
isLoading boolean

The state indicating if the applicants are being fetched.
onBack function

The function to call when the back button is clicked.
onNext function

The function to call when the next button is clicked.
onPrev function

The function to call when the previous button is clicked.
hasMore boolean

The state indicating if there are more applicants to fetch.
currentPage number

The current page number.
jobId string

The ID of the job for which applicants are being displayed.
onViewApplication function

The function to call when viewing an applicant's details.
selectedApplicant Object

The currently selected applicant's details.
onCloseApplicationDetails function

The function to call when closing the application details modal.

    It shows the applicant's avatar, name, location, and allows updating the application status and notes.

Example
<ViewApplicationDetailsModal
  application={{ avatar: "avatar.jpg", name: "John Doe", location: "New York" }}
  status="Pending"
  notes="Initial review completed."
  isUpdating={false}
  isLoading={false}
  onClose={() => {}}
  onStatusChange={(newStatus) => {}}
  onNotesChange={(newNotes) => {}}
  onSave={() => {}}
  open={true}
/>
