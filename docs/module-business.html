The BusinessPageContainer component serves as the container component for the business page.
It manages the state and logic for fetching user companies, handling pagination, and navigating
to the company creation form. It passes the necessary data and handlers to the presentation
component BusinessPagePresentation.

Description:

  • The BusinessPageContainer component serves as the container component for the business page.
    It manages the state and logic for fetching user companies, handling pagination, and navigating
    to the company creation form. It passes the necessary data and handlers to the presentation
    component BusinessPagePresentation.

Properties:
Name Type Description
companies Array

The list of companies fetched from the API.
isLoading boolean

Indicates whether the data is currently being loaded.
isError boolean

Indicates whether there was an error during data fetching.
error string

The error message, if any, encountered during data fetching.
currentPage number

The current page number for pagination.
onPageChange function

Callback function to handle page changes.
onCreateCompany function

Callback function to navigate to the company creation form.
Example
<BusinessPageContainer />

The rendered BusinessPageContainer component.
